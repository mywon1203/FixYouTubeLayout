# Fix YouTube Player Layout

Restores YouTube's watch page layout to its previous behavior when the video is shifted too far to the left.

## Install

1. Open `chrome://extensions`.

2. Enable Developer Mode.

3. Click `Load unpacked`.

4. Select this extension folder.

## What it does

- Displays a guide panel to the left of the video, just like the previous layout.

## Notes

- This extension was written with Codex and may stop working if YouTube's DOM structure changes.

- This extension only changes YouTube watch pages. Home and other pages keep YouTube's default layout.
